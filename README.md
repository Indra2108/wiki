# Wiki
This is my personal wiki. build with Docusaurus.

Live: https://indrdmr.haliya.net

## How to Run/Build
```
git clone https://github.com/Indra2108/wiki.git
npm install
npm start
```
If you want to build this project
```
npm run build
```

In case you want to test your build locally
```
npm run serve
```
